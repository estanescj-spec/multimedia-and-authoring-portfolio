Multimedia & Authoring Portfolio

This website is a collection of my multimedia outputs for the subject Multimedia and Authoring. It includes 3D animation using Blender and frame-by-frame rotoscope animation work.

What This Includes

• Blender Solar System Animation

• Age Calculator (Planet Time Conversion)

• Rotoscope Live Action Animation

• Weekly Progress Documentation (12 Weeks)

Purpose

This project demonstrates understanding of animation, timing, visual storytelling, and multimedia production workflows.
